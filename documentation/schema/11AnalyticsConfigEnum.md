# 20. ANALYTICS & TELEMETRY

## 20.1 Analytics Events

```sql
TABLE: analytics_events
├── id: UUID [PK]
├── session_id: UUID
├── player_id: UUID
├── character_id: FK -> characters
├── occurred_at: TIMESTAMP
│
├── ## EVENT
├── event_type: VARCHAR(50)
├── event_category: VARCHAR(30)
├── event_name: VARCHAR(100)
├── event_data: JSONB
│
├── ## CONTEXT
├── location_id: FK -> locations
├── mission_id: FK -> missions
├── coordinates: POINT
│
├── ## SESSION
├── session_time_seconds: INT
├── playtime_total_seconds: BIGINT
│
├── ## META
├── client_version: VARCHAR(20)
├── platform: VARCHAR(30)
└── is_debug: BOOLEAN
```

## 20.2 Play Sessions

```sql
TABLE: play_sessions

├── deliveries_completed: INT
├── credits_earned: INT
├── credits_spent: INT
├── xp_earned: INT
├── distance_traveled_km: DECIMAL(10,2)
├── enemies_defeated: INT
├── deaths: INT
│
├── ## PROGRESSION
├── tier_at_start: INT
├── tier_at_end: INT

├── rating_at_end: DECIMAL(7,3)

├── ## META
├── client_version: VARCHAR(20)
├── platform: VARCHAR(30)
└── session_quality_score: INT [0-100]
```
