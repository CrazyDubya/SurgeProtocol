## Implementation Notes

### Readability
- Text visible at close range
- Optional "read" interaction for longer pieces
- Font matches location (corporate = clean, hollows = rough)

### Discovery Rewards
- Some text provides hints for hidden content
- Easter eggs tracked for achievement
- Lore collectibles possible expansion

### Updates
- Text can reflect story state (post-strike graffiti changes)
- Player name appears in appropriate contexts
- Date-specific content for immersion
