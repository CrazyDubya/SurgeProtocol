# Package Descriptions Catalog

## Purpose

## PACKAGE CONDITION MODIFIERS

### Damage States
- **Pristine**: Full payment
- **Minor Wear**: 90% payment
- **Damaged**: 50% payment
- **Destroyed**: No payment + reputation hit

### Environmental Factors
- **Wet**: Affects documents, electronics
- **Cold**: Affects bio samples, food
- **Heat**: Affects chemicals, organics
- **Shocked**: Affects electronics, fragile items

## IMPLEMENTATION NOTES

### Selection by Tier
- Tier 0-2: Packages 1-10 (legal only)
- Tier 2-4: Packages 1-20 (gray market opens)
- Tier 4-6: Packages 11-30 (high risk available)
- Tier 6-8: Packages 21-40 (extreme content)

### Combination Guidelines
- Match package risk to client nervousness
- Higher value packages = higher complications
- Some packages open special dialogue options
- Examination is optional player action
